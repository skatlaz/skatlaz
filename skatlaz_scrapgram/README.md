# 🚀 Skatlaz Scrapgram

A complete messaging system inspired by Telegram.

## 🔥 Features

* Real-time chat (WebSocket)
* REST API
* File upload
* Giphy
* Anti-spam
* Admin panel
* Basic encryption

## ▶️ Run

```bash
pip install -r requirements.txt
python app.py
```

## 🐳 Docker

```bash
docker-compose up
```
